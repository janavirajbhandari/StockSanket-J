# StockSanket - NEPSE Stock Forecasting Web App

A machine learning web application that forecasts Nepal Stock Exchange (NEPSE) 
stock prices using LSTM neural networks and real-time data.

## Features
- Stock price prediction using LSTM deep learning model
- Real-time NEPSE data via NepseAPI
- Tested on NABIL Bank and NIMB stocks
- Automated daily data updates via GitHub Actions
- Web interface with authentication

## Tech Stack
Python, LSTM, Flask, NepseAPI, Pandas, Matplotlib, GitHub Actions

## Models
- LSTM-based time series forecasting
- Trained and tested on NABIL, NIMB stock data
